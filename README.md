# Oxford-IIIT Pet Classifier 🐾

Projet de **classification d’images de races de chats et chiens** à partir du dataset **Oxford-IIIT Pet**. Ce projet montre mes compétences en **Deep Learning, CNN, et gestion de dataset multi-classes**.

---

## 🔹 Objectifs

- Construire un **CNN performant** pour identifier la race d’un animal.  
- Gérer un dataset multi-classes et non équilibré.  
- Fournir un pipeline complet pour l’entraînement et la prédiction.

---

## 🔹 Réalisations clés

- Prétraitement et augmentation d’images pour améliorer la robustesse du modèle.  
- CNN optimisé pour la classification multi-classes.  
- Précision élevée sur le test set et visualisation des prédictions.

---

## 🔹 Technologies

**Python**, **TensorFlow/Keras**, **CNN**, **image augmentation**, **multi-class classification**.
